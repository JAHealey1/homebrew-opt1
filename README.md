# homebrew-opt1

[Homebrew tap](https://docs.brew.sh/Taps) for [Opt1](https://github.com/JAHealey1/Opt1-Releases),
a free macOS helper for *RuneScape* clue scrolls.

## Install

```bash
brew tap JAHealey1/opt1
brew install --cask opt1
```

## Update

Opt1 ships with [Sparkle](https://sparkle-project.org), so installed apps
update themselves in the background within ~24h of a new release. You can
also force an immediate check from the menu bar: *Opt1 → Check for
Updates…*.

If you'd rather drive updates through Homebrew itself, you can:

```bash
brew update
brew upgrade --cask opt1
```

`brew upgrade` won't fight Sparkle — the cask is declared
`auto_updates true`, so brew only tracks new cask file revisions
(published when a new tag is cut on the source repo).

## Uninstall

```bash
brew uninstall --cask opt1
```

To also remove preferences, caches, logs, and any data Opt1 wrote to
`~/Library/Application Support/Opt1`:

```bash
brew uninstall --zap --cask opt1
```

## What's in this repo

A single Cask file: [`Casks/opt1.rb`](Casks/opt1.rb). It points at the
latest notarized DMG hosted on
[`JAHealey1/Opt1-Releases`](https://github.com/JAHealey1/Opt1-Releases),
pinned by `version` and `sha256`. The file is regenerated by CI on every
new Opt1 release, so the tap is always in sync with the latest GitHub
Release.

## Where Opt1 actually lives

- **Releases / DMGs / Sparkle appcast:**
  [`JAHealey1/Opt1-Releases`](https://github.com/JAHealey1/Opt1-Releases)
- **Bug reports:**
  [`Opt1-Releases/issues`](https://github.com/JAHealey1/Opt1-Releases/issues/new)
- **Source:** closed.

This tap is intentionally a thin redirect — please don't open issues
about Opt1 itself here.

## System requirements

- macOS 14.0 (Sonoma) or newer
- Apple Silicon or Intel (universal binary)
